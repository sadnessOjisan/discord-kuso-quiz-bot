# discord-kuso-quiz-bot

just [kuso quiz](https://twitter.com/hashtag/kuso%E3%81%AA%E3%81%9E%E3%81%AA%E3%81%9E?src=hashtag_click&f=live) bot.

```sh
rustup toolchain install beta

cargo init --edition 2021
```

Run

```
cargo +beta run
```

and open discord.
